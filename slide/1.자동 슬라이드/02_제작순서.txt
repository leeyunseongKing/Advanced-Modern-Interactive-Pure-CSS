기본 html 작업 부터 작성을 하고 시작한다.
header, section을 나눈다.
디자인을 보면서 div로 나누어야 할 곳들을 상위 div에서 하위 div까지 만든다.
그리고 줄바꿈이 일어나면 안되고 정렬해야 하는 곳들은 미리 display: flex 작업을 할 곳들을 class를 넣어 둔다.
html작업을 끝낸 뒤에 css 작업을 시작한다.
css 작업은 제활용 하거나 전역으로 사용하는 것을 상위에 두고 뭔저 만든다.
그리고 큰 틀을 하나씩 잡으면서 상위에서 하위로 내려오면서 css작업을 한다.
animation 효과를 넣고 기능 만든다.
마지막으로 font를 하나씩 적용하면서 검토하고 끝낸다.
